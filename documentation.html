Complete Project Technical Documentation & Source Code

Author: Siddharth Pawar & Athrav Nikam

1. Ticket Generation Workflow

This project uses a client-side state management approach to handle bookings without a backend server.

1
Data Entry: User fills booking.html form. JavaScript listens for the 'submit' event.
2
Dynamic Inputs: As 'Number of Travelers' changes, JS creates name fields for each passenger.
3
Price Engine: Real-time calculation based on Destination Price × Travelers + Car Surcharges.
4
Storage: Data is bundled into a JSON object and saved to localStorage.setItem('latestBooking', ...).
5
Rendering: ticket.html reads the data. If 'Flight', it loops through names to create multiple tickets. If 'Car', it generates one Group Pass with driver details.
